# CHANGELOG

## [0.1.0] - 2026-02-10

### Added

- Initial release of the Code Execution MCP Server
- `execute_terminal` tool for running shell commands
- `execute_python` tool for running Python code via IPython
- `get_output` tool for retrieving session output
- `reset_terminal` tool for resetting sessions
- Multi-session support with persistent state
- Configurable timeouts via environment variables
- Smart output handling with prompt and dialog detection
- File logging support
